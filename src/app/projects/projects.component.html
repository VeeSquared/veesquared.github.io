RegistrarNow!

RegistrarNow! is a web application which performs as a basic university registration system for Administrators, Professors, and Students. The application allows students to send and receive messages from Academic Advising. Users can login, enroll in classes, view course listings, update personal info, view grades with their class rank, class grade distribution and final schedule.

SHA-1 Hash Algorithm

A simulated design of the SHA-1 Hash Algorithm developed in Verilog. The simulated design takes input of a message and computes it into a 160-bit SHA1 hash. The simulated design utilizes multiple pipelines to increase performance of the algorithm and reduce area delay between the execution of each instruction.

BeanMe!

AdminALL

AdminALL is a web app created for Administrators to be able to manage users. There are three user groups which include Administrators, Group Level Users, and General Users. The Admins are allowed to create, edit, and delete user profiles. If a user forgets their password, they can send a request have their password reset.

Dopamemes

Dopamemes is a web app which allows users to create, upload, and organize memes. There is also a chrome extension that complements this web application. This web app features saving memes in albums, sorting memes, and rating memes. Users can also drag and drop and upload memes from their desktop and save it to their only library.

Boggle

Implemented the backend logic to find all possible words for a game of Boggle. The logic utilizes a Multi-way trie to maintain a lexicon and a Depth First Search to find all valid words on a randomly generated 4x4 boggle board.
